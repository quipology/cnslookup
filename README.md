# cNSLookup v0.0.1

## What's this?
This is a tool that will do a nslookup on a provided list of hosts and save the results to CSV file.

## Usage
```
Usage of ./cnslookup:
  -f string
        Filename of hosts file.
  -o string
        Name of output file. (default "results.csv")
```
